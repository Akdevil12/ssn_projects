Welcome to Aiport Management System version 1.
--------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------
This version contains the following features:-
	1. add flights
	2. Check the departure List
	3. Check the arrival list
	4. emergency and contact information for flight operators
	5. Special feature of ADMIN to edit the flight details
	6. Registration for the flight operators
	7. disabled option of update flight status for flight operators

--------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------
The following needs to be present with the following configurations:-
	1. MYSQL Server 5.5
		-> It needs to have localhost as "3306"
		-> It needs to have username as "root"
		-> It needs to have password as "admin"
	2. JDK version atleast 1.8
--------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------
After getting the required software
copy the "airport.sql" file to the following path--> "C:\Program Files\MySQL\MySQL Server 5.5\bin"
run the following command in MYSQL server

	mysql> create database airport_management;
	mysql>use airport_management;
	mysql>source airport.sql

--------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------
Then Execute the airport_management.jar file
